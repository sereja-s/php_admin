1- функция php: substr()- возвращает подстроку строки (1-ый параметр), начинающейся с № символа (2-ый параметр) и длиной кол-во символов (3-ый параметр)

2- функция php: strrpos()- ищет последнее вхождение подстроки (2-ый параметр) в строке (1-ый параметр) и возвращает номер позиции

3- функция php: rtrim(), которая обрезает концевые пробелы в начале и конце строки (1-ый параметр), а также символы в конце строки, которые указаны в качестве второго не обязательного параметра на входе 

4- функция php: strlen() показвает (возвращает) длину строки (массива символов)

5- функция php: explode(string $separator, string $string, int $limit = PHP_INT_MAX): array  
Возвращает массив строк, полученных разбиением строки string с использованием separator в качестве разделителя.

6- функция php: ucfirst(string $string)
Возвращает строку string, в которой первый символ переведён в верхний регистр, если этот символ является буквой

7- функция php: array_shift(array &$array)
Извлекает первое значение массива array и возвращает его, сокращая размер array на один элемент.
Все числовые ключи будут изменены таким образом, что нумерация массива начнётся с нуля, в то время как строковые ключи останутся прежними

8- функция php: is_dir(string $filename) Определяет, является ли имя файла директорией На вход подаётся полный путь к папке
