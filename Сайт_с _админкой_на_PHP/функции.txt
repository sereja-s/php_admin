 функция php: substr()- возвращает подстроку строки (1-ый параметр), начинающейся с № символа (2-ый параметр) и длиной кол-во 	символов (3-ый параметр)

 функция php: strrpos()- ищет последнее вхождение подстроки (2-ый параметр) в строке (1-ый параметр) и возвращает номер позиции

 функция php: rtrim(), которая обрезает концевые пробелы в начале и конце строки (1-ый параметр), а также символы в конце 			строки, которые указаны в качестве второго не обязательного параметра на входе 

 функция php: strlen() показвает (возвращает) длину строки (массива символов)

функция php: explode(string $separator, string $string, int $limit = PHP_INT_MAX): array  
	Возвращает массив строк, полученных разбиением строки string с использованием separator в качестве разделителя.

функция php: ucfirst(string $string)
	Возвращает строку string, в которой первый символ переведён в верхний регистр, если этот символ является буквой

функция php: array_shift(array &$array)
	Извлекает первое значение массива array и возвращает его, сокращая размер array на один элемент.
	Все числовые ключи будут изменены таким образом, что нумерация массива начнётся с нуля, в то время как строковые ключи 	останутся прежними

функция php: is_dir(string $filename) Определяет, является ли имя файла директорией На вход подаётся полный путь к папке

функция php: count($url) посчитаем кол-во элементов массива $url (переданного в параметры на вход)

функция php: extract() разбирает массив параметров, поданных на вход  В памяти текущей символьной таблицы создаёт переменные 		(вида: ключ => значение) из массива, поданного на вход в области видимости функции т.е.доступные только внутри функции, в 		которой используется ф-ия: extract()

функция php: str_replace — Заменяет все вхождения строки поиска на строку замены

функция php: ob_start- которая открывает текущий буфер обмена (пока активна буферизация вывода, скрипт не отправляет вывод 		(кроме заголовков), вместо этого вывод сохраняется во внутреннем буфере)

функция php: ob_get_clean()- возвращает данные из буфера обмена и закрывает (удаляет его)

функция php: method_exists() — Проверяет, существует ли метод в данном классе

функция php: func_get_arg()- Возвращает указанный аргумент из списка аргументов пользовательской функции
